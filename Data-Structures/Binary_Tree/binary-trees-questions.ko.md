## 1. `identical`

두 개의 이진 트리 `tree1`, `tree2`가 구조적으로 동일한지 판별하는 재귀 C 함수 `identical()`을 작성하시오. 두 트리가 구조적으로 동일하면 `1`, 그렇지 않으면 `0`을 반환해야 합니다. 두 이진 트리가 모두 비어 있거나, 둘 다 비어 있지 않으면서 왼쪽과 오른쪽 서브트리가 서로 같은 값과 같은 구조로 배치되어 있으면 구조적으로 동일하다고 봅니다.

함수 원형:

```c

```

## 5. `mirrorTree`

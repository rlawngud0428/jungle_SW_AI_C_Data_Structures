## 1. `levelOrderTraversal`

큐를 사용하여 이진 탐색 트리를 레벨 순서대로 순회하는 반복형 C 함수 `levelOrderTraversal()`을 작성하시오. 순회는 루트 레벨부터 시작해야 합니다. 큐에서 원소를 넣거나 뺄 때는 `enqueue()`와 `dequeue()`만 사용해야 합니다. 큐가 비어 있지 않다면, 시작할 때 먼저 큐를 비워야 합니다.

함수 원형:

```c
void levelOrderIterative(BSTNode *root);
```

예를 들어, 그림 1의 이진 트리에 대해 레벨 순회 결과는 `20, 15, 50, 10, 18, 25, 80`입니다.

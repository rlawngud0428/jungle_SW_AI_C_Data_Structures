## 1. `createQueueFromLinkedList`

`createQueueFromLinkedList()`라는 C 함수를 작성하시오. 이 함수는 연결 리스트에 저장된 모든 정수를 순서대로 `enqueue`하여, 연결 리스트 기반 큐를 만들어야 합니다. 연결 리스트의 첫 번째 노드가 먼저 들어가고, 그다음 두 번째 노드가 들어가는 방식입니다. 큐가 비어 있지 않다면, 시작할 때 먼저 큐를 비워야 합니다.

함수 원형:

```c
void createQueueFromLinkedList(LinkedList *ll , Queue *q);
```

예시 입력/출력:

```text
결과 연결 리스트: 1 2 3 4 5
선택을 입력하세요(1/2/3/0): 2
결과 큐: 1 2 3 4 5
```

## 2. `createStackFromLinkedList`

`createStackFromLinkedList()`라는 C 함수를 작성하시오. 이 함수는 연결 리스트에 저장된 모든 정수를 순서대로 `push`하여, 연결 리스트 기반 스택을 만들어야 합니다. 연결 리스트의 첫 번째 노드가 먼저 `push`되고, 그다음 두 번째 노드가 `push`되는 방식입니다. 스택이 비어 있지 않다면, 시작할 때 먼저 스택을 비워야 합니다.

함수 원형:

```c
void createStackFromLinkedList(LinkedList *ll , Stack *stack);
```

예시 입력/출력:

```text
결과 연결 리스트: 1 3 5 6 7
선택을 입력하세요(1/2/3/0): 2
결과 스택: 7 6 5 3 1
```

## 3. `isStackPairwiseConsecutive`

`isStackPairwiseConsecutive()`라는 C 함수를 작성하시오. 이 함수는 스택 안의 숫자들이 두 개씩 짝을 이루어 연속된 수인지 검사해야 합니다. 정수를 스택에 넣거나 뺄 때는 `push()`와 `pop()`만 사용해야 합니다.

함수 원형:

```c
int isStackPairwiseConsecutive(Stack *s);
```

예시 입력/출력:

```text
예를 들어, 스택이 (16, 15, 11, 10, 5, 4)인 경우:
스택은: 16 15 11 10 5 4
스택의 각 쌍은 연속된 수입니다.

예를 들어, 스택이 (16, 15, 11, 10, 5, 1)인 경우:
스택은: 16 15 11 10 5 1
스택의 각 쌍은 연속된 수가 아닙니다.

예를 들어, 스택이 (16, 15, 11, 10, 5)인 경우:
스택은: 16 15 11 10 5
스택의 각 쌍은 연속된 수가 아닙니다.
```

## 4. `reverseQueue`

`reverseQueue()`라는 C 함수를 작성하시오. 이 함수는 스택을 이용하여 큐를 뒤집어야 합니다. 스택에서 정수를 넣거나 뺄 때는 `push()`와 `pop()`만 사용해야 하고, 큐에서 정수를 넣거나 뺄 때는 `enqueue()`와 `dequeue()`만 사용해야 합니다. 스택이 비어 있지 않다면, 시작할 때 먼저 스택을 비워야 합니다.

함수 원형:

```c
void reverseQueue(Queue *q);
```

예를 들어, 큐가 `(1, 2, 3, 4, 5)`라면 결과 큐는 `(5, 4, 3, 2, 1)`이 됩니다.

## 5. `recursiveReverseQueue`

정수 큐에 저장된 항목들의 순서를 뒤집는 재귀 C 함수 `recursiveReverseQueue()`를 작성하시오.

함수 원형:

```c
void recursiveReverseQueue(Queue *q);
```

예를 들어, 큐가 `(1, 2, 3, 4, 5)`라면 결과 큐는 `(5, 4, 3, 2, 1)`이 됩니다.

## 6. `removeUntilStack`

`removeUntilStack()`라는 C 함수를 작성하시오. 이 함수는 스택에서 지정한 값이 처음 나타날 때까지 모든 값을 `pop`해야 합니다.

함수 원형:

```c
void removeUntilStack(Stack *s, int value);
```

맨 위 원소가 왼쪽에 표시된 스택 `(1, 2, 3, 4, 5, 6, 7)`에 대해 `value = 4`로 `removeUntilStack()`을 호출하면 결과 스택은 `(4, 5, 6, 7)`이 됩니다. 마찬가지로, 스택 `(10, 20, 15, 25, 5)`에 대해 `value = 15`로 호출하면 결과 스택은 `(15, 25, 5)`가 됩니다.

## 7. `balanced`
